# ⚛️ React Learning Module

This section documents hands-on learning and practice of core React concepts through structured examples and mini tasks.

Each folder represents a focused concept with implementation to reinforce understanding.

---

## 📌 What This Covers

- Component-based architecture  
- Props and data flow  
- Event handling  
- State management using Hooks  
- Conditional rendering  
- List rendering  
- Form handling  
- Advanced Hooks (useEffect, useContext, useRef)
